# Slottet

Slottet er et eksamensprojekt udviklet i ASP.NET Core og Blazor Web App.

## Arkitektur
Løsningen består af følgende projekter:

- Slottet.Web – Blazor Web App frontend
- Slottet.Api – backend API
- Slottet.Application – applikationslogik og use cases
- Slottet.Domain – domænemodeller og forretningsregler
- Slottet.Infrastructure – database, persistence og eksterne integrationer

## Teknologier
- .NET 10
- Blazor Web App
- ASP.NET Core Web API
- Entity Framework Core
- Azure SQL Database
